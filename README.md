## 📌 소개
이 레포지토리는 Django, React, Next.js를 활용하여 풀스택 개발을 공부한 내용을 정리하고 공유하는 공간입니다. 백엔드와 프론트엔드를 함께 다루며, 학습 과정에서 배운 개념, 예제 코드, 프로젝트 등을 기록합니다.

## 🚀 기술 스택
- **Backend**: Django, Django REST Framework (DRF)
- **Frontend**: Node.js, React, Next.js
